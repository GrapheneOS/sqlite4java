sqlite4java is a minimalistic high-performance Java wrapper for SQLite database.

Main classes are {@link SQLiteConnection}, {@link SQLiteStatement} and {@link SQLite}. Here's the typical use: SQLiteConnection db = new SQLiteConnection(new File("/tmp/database")); db.open(true); SQLiteStatement st = db.prepare("INSERT INTO table VALUES (?, ?, ?)"); st.bind(1, "value").bind(2, 123L).bind(3, 3.141); st.step(); st.dispose(); db.close();

See http://code.google.com/p/sqlite4java for more information.
